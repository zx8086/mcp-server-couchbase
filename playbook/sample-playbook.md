# Sample Playbook

This is a sample playbook that demonstrates the functionality of the playbook resource system.

## Overview

This playbook provides an example of how to use the playbook system in your MCP server.

## Features

- Markdown formatting support
- Automatic resource registration
- Directory listing
- Individual playbook access

## Usage

You can access this playbook through the MCP server's resource system using the URI: `playbook://sample-playbook`
